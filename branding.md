# Forest-Style Brand Reference

A quick-reference guide to the forest-style brand system. For the full AI-instructional version, see [`skills/forest-style.md`](./skills/forest-style.md).

---

## Colors

| Role | Name | Hex |
|------|------|-----|
| Primary | Forest Green | `#1A5C38` |
| Accent | Canyon Red | `#C0392B` |
| Sky | Oregon Blue | `#4A9CC7` |
| Earth | Meadow Green | `#5A8A3C` |
| Background | Warm Cream | `#F5F0E8` |
| Surface | White | `#FFFFFF` |
| Text Primary | Deep Forest | `#1A2E1A` |
| Text Secondary | Slate | `#5C6B5C` |
| Border | Soft Sage | `#C8D8C0` |

**Key rules:** Forest Green anchors every layout. Warm Cream replaces pure white for backgrounds. Canyon Red punctuates — never dominates. Oregon Blue is for links and interactive elements only.

---

## Typography

| Level | Font | Weight |
|-------|------|--------|
| Headings (H1–H3) | Montserrat | 600–700 Bold |
| Display / Logo | Montserrat | 800 ExtraBold |
| Body | Open Sans | 400 Regular |
| Buttons / Labels | Montserrat | 700 Bold, uppercase |
| Captions | Open Sans | 400 Regular |

**Key rules:** Montserrat for all headings — always title case or uppercase, never lowercase. Open Sans for body text. No decorative or script fonts.

## Spacing

| Token | Value |
|-------|-------|
| `space-xs` | 4px |
| `space-sm` | 8px |
| `space-md` | 16px |
| `space-lg` | 24px |
| `space-xl` | 40px |
| `space-2xl` | 64px |

**Border radius:** `6px` cards/inputs · `4px` buttons · `999px` pills · `0px` tables

---

## Quick Component Reference

**Buttons** — Primary: Forest Green bg, white text. Secondary: white bg, Forest Green border/text. Danger: Canyon Red. All buttons: Montserrat 700 uppercase, `4px` radius, `10px 20px` padding.

**Cards** — White bg, `1px #C8D8C0` border, `6px` radius, `24px` padding, subtle Forest Green shadow.

**Tables** — Forest Green header row, alternating white/warm rows, Soft Sage borders, `12px 16px` cell padding.

**Badges** — Pill shape (`999px`), Montserrat 600 11px uppercase. Green / Red / Blue / Neutral variants.

**Alerts** — 4px left border, light-tinted background. Info: Oregon Blue. Success: Meadow Green. Error: Canyon Red.

**Nav** — Forest Green background, white Montserrat text, Canyon Red underline on active item.

## Voice & Tone

- Warm and community-first — write for a neighbor, not a customer
- Action-oriented — "Connect," "Build," "Grow"
- Grounded — reference the place and people when relevant
- Concise — short sentences, no filler

---

## Imagery

- Pacific Northwest landscapes: Cascades, high desert, rivers, forests
- Warm/natural light — avoid cold blue filters
- Diverse, authentic people — no polished stock photography
- Text-over-photo overlay: `rgba(26, 46, 26, 0.45)` Forest Green
